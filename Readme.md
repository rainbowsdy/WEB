# Projet Web site de gambling

## Frontend

Aller dans le répertoire `view`

Pour un premier démarrage faire la commande 

```shell
npm install
```
Cela va installer toutes les librairies dont on a besoin (qui sont listées dans le fichier `package.json`)

Ensuite à chaque fois que l'on veut démarrer le serveur de frontend faire:

```shell
npm run start
```
